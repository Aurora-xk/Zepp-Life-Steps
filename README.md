# 小米运动步数修改工具

这是一个用于修改小米运动步数的 Web 应用。

## 功能特点

- 简洁的用户界面
- 支持自定义步数
- 一键部署到 Vercel

## 本地开发

1. 安装依赖：
```bash
npm install
```

2. 运行开发服务器：
```bash
npm run dev
```

3. 打开浏览器访问 http://localhost:3000

## 部署到 Vercel

1. 将代码推送到 GitHub 仓库
2. 在 Vercel 中导入该仓库
3. Vercel 将自动部署你的应用

## 注意事项

- 请合理使用，不要频繁修改步数
- 建议每次修改的步数不要超过合理范围
- 请妥善保管您的账号密码
